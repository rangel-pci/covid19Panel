<h2>install & run</h2>

```bash
#Clone the repository
$ git clone https://github.com/rangel-pci/covid19Panel.git

#Enter the repository folder
$ cd covid19Panel

#Install the dependencies
$ npm install

#Run the application
$ npm start

#Open in http://localhost:3000/
```
